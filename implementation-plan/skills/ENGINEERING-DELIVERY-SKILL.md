# Skill: Engineering Delivery

## Purpose
Ensure predictable engineering throughput with quality safeguards.

## Inputs

- Assigned engineering issues
- Release checklist
- Defect and reliability signals

## Procedure

1. Select highest-priority unblocked issue.
2. Plan minimal viable implementation slice.
3. Implement and validate locally.
4. Record implementation notes and risks.
5. Move issue to review with acceptance evidence.

## Output

- Completed issue increments
- Updated technical notes
- Clear blocker escalations
