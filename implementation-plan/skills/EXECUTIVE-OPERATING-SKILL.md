# Skill: Executive Operating System

## Purpose
Guide CEO-level execution in Paperclip with goal alignment, governance, and budget control.

## Inputs

- Dashboard metrics
- Blocked issues
- Approval queue
- Cost trend by agent

## Procedure

1. Review goal progress (G1/G1.x).
2. Identify critical blockers and assign owners.
3. Approve/reject pending governance actions.
4. Reprioritize top 10 issues for next 24h.
5. Publish daily strategic directive comment.

## Output

- Priority list
- Delegation actions
- Governance decisions
