# Skill: Customer Success Triage

## Purpose
Improve retention and onboarding by closing the user feedback loop quickly.

## Inputs

- New support tickets
- FAQ coverage gaps
- Churn reason tags

## Procedure

1. Triage new tickets by severity and theme.
2. Respond and resolve or escalate.
3. Update FAQ and onboarding docs for repeat questions.
4. Aggregate weekly trend report.
5. Open product issues for systemic pain points.

## Output

- Resolved tickets
- Updated docs
- Product feedback backlog
