# Skill: Growth Experiment Loop

## Purpose
Run rapid, measurable growth experiments with clean learning loops.

## Inputs

- ICP matrix
- Active channel metrics
- Current experiment backlog

## Procedure

1. Pick one hypothesis with measurable KPI.
2. Launch variant with controlled variables.
3. Monitor early signal window.
4. Decide kill/iterate/scale.
5. Document learnings and next test.

## Output

- Experiment report
- Channel optimization recommendations
